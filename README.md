# Biyahe Now - Van Booking System

A comprehensive bus booking and management system built with PHP, MySQL, and JavaScript.

## Features

- **Passenger Dashboard**: Book trips, track live location, view trip history
- **Driver Dashboard**: Manage trips, check passenger lists, live tracking
- **Admin Dashboard**: Manage drivers, routes, and system operations
- **Real-time Tracking**: Road-to-road navigation using Leaflet maps
- **Mobile Responsive**: Works on all devices

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript, Leaflet.js
- **Backend**: PHP 8.0+
- **Database**: MySQL
- **Maps**: OpenStreetMap with Leaflet Routing Machine

## Installation

1. Clone the repository
2. Set up database using the provided SQL files
3. Configure database connection in `Php/db.php`
4. Run on local server (XAMPP, WAMP, etc.)

## Deployment

Compatible with:
- Render
- Heroku
- Any PHP hosting service

## Routes Available

- Naval → Tacloban (6:00 AM, 10:00 AM)
- Naval → Ormoc (6:00 AM, 10:00 AM)
